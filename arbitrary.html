Example Page

Pretty much everything on this page can be changed on the fly using an external file.

Instructions

Below are the instructions for the current project:

  • Write some javascript in ./yours.js to stick some draggable image/text/video components in #container as pulled from an external data source of your choice.
  • Add a title and some simple formatting to each component as defined *once* in an arbitrary html file in the same directory using a single GET.
  • Clicking on one of the components should make it smoothly grow to fill the whole screen avoiding image pixelation.
  • Then when you click on it again it should shrink back to the same position it was at before you clicked on it.
  • Also add a button to randomly rearrange the components within #container.
